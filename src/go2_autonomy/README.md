# go2_autonomy

Autonomous navigation and perception packages.

## Packages

| Package | Purpose |
|---------|---------|
| `go2_autonomy` | High-level autonomous behaviours and mission logic |
| `go2_slam` | Simultaneous localisation and mapping |
| `go2_localisation` | Localisation against existing maps (map → odom TF) |
| `go2_navigation` | Nav2 integration and path planning |
| `go2_perception` | Object detection and scene understanding |
