# go2_hardware

Sensor and hardware interface packages.

## Packages

| Package | Purpose |
|---------|---------|
| `go2_sensors` | Sensor configuration and parameter management |
| `go2_lidar` | Unitree L2 LiDAR processing (PointCloud filtering, 2D scan extraction) |
| `go2_camera` | Front camera calibration and ZED camera integration |
